1) Paris

in the file showing rental market price, i have the geojson of 80 regions inside paris. which is plenty to have more accuracy on data (on reserve that i have sufficient datapoints per area).
If not the case I will use the arrondissement geojson, and get a coarser grid details.

Listing airbnb:
On commence par charger les données des quartiers de Paris depuis le fichier paris_neighborhoods.csv. La géométrie de chaque quartier, stockée au format GeoJSON, est convertie en polygone.
Ensuite, on charge les données des logements Airbnb depuis paris_airbnb.csv. On utilise les colonnes de latitude et de longitude pour créer un point géographique pour chaque annonce.
Chaque point est ensuite associé au quartier dans lequel il se trouve grâce à une jointure spatiale. Cela permet de savoir combien d’annonces Airbnb se trouvent dans chaque quartier.
On regroupe les données par quartier pour compter le nombre total d’annonces.
Enfin, on affiche une carte de Paris avec les quartiers colorés selon le nombre d’annonces Airbnb qu’ils contiennent.

Listing des locations paris.
On commence par charger les quartiers de Paris depuis le fichier paris_neighborhoods.csv. La colonne contenant les identifiants est renommée, et les géométries stockées au format GeoJSON sont converties en objets de type polygone utilisables par GeoPandas.
Ensuite, on lit les données de locations depuis paris_rentals.csv. Les coordonnées GPS sont extraites depuis une colonne qui les contient sous forme de texte. Ces coordonnées sont ensuite transformées en points géographiques. On récupère également le prix au mètre carré depuis une autre colonne.
Chaque point de location est ensuite associé au quartier dans lequel il se situe grâce à une jointure spatiale. Cela permet de savoir à quel polygone chaque point appartient.
On regroupe ensuite les données par quartier pour calculer le prix moyen de location au mètre carré dans chaque zone.
Enfin, on affiche une carte de Paris où chaque quartier est coloré selon son prix moyen, ce qui permet de visualiser les différences de loyers sur l’ensemble du territoire.
